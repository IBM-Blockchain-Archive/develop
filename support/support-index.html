Get Support

IBM Support

IBM Blockchain provides a fully integrated platform designed to accelerate developing, governing, and operating a multi-organization business network. You can access support using the Platform via the following options:

  • If you use the IBM Blockchain Platform service on IBM Cloud, you can raise a ticket via IBM Cloud Support.
  • IBM Premium Blockchain Support is available by request.

Before seeking support, please ensure you are running a Supported Operating Environment by reviewing the statement at the bottom of this page.

Talk to the Community

For those who do not have a paid support plan, here are some options for engaging with other developers in the community:

  • Find learning materials on developerWorks
  • Ask a question on StackOverflow
  • Discuss with experts on Rocket.Chat

Supported Operating Environments

IBM Blockchain Platform

The Develop component of the IBM Blockchain Platform is supported for use with and deployment to all IBM Blockchain Platform environments.

Operating systems

The following operating systems are supported for use with IBM Blockchain Platform: Develop.

Operating system Architecture(s) Version(s)
Ubuntu Linux x86-64 16.04 LTS
macOS x86-64 Sierra (10.12), High Sierra (10.13)

These operating systems are supported for use when installing IBM Blockchain Platform: Develop components using either npm modules or Docker images.

Cloud environments

The following cloud environments are supported for use with IBM Blockchain Platform: Develop components.

Platform/cloud Service Comments
IBM Cloud Container Service IBM-signed Docker images must be used. Sample Kubernetes scripts are provided.

Docker

The following versions of Docker are supported for use with the Docker images for IBM Blockchain Platform: Develop, as long as Docker is running on a supported operating system or cloud environment.

Component Architecture(s) Version(s)
Docker Engine x86-64 17.03 or higher
Docker Compose x86-64 1.8 or higher

Node.js and npm

The following versions of Node.js and npm are supported for use with the npm modules for IBM Blockchain Platform: Develop, as long as Node.js and npm are running on a supported operating system or cloud environment:

Component Architecture(s) Version(s)
IBM SDK for Node.js x86-64 v8.9.0.0 or higher
npm x86-64 v5.5.1 or higher

IBM SDK for Node.js can be downloaded from IBM: https://developer.ibm.com/node/sdk/v8/

Please note that the IBM SDK for Node.js includes a supported version of npm, and npm does not need to be installed separately.

Web browsers

The following web browsers are supported for use with the web components of IBM Blockchain Platform: Develop; including the playground, the REST API explorer, and the generated web application:

Browser Mozilla Firefox Google Chrome Safari
Version(s) 57 or higher 62 or higher 11 or higher

Mobile web browsers, including mobile variants of the above browsers, are not currently supported.

Editor plugins

IBM Blockchain Platform: Develop plugins are provided and supported for the following editors and versions:

Editor Version(s) Plugin link
Atom v1.20 or higher Hyperledger Composer package
Visual Studio Code v1.16 or greater Hyperledger Composer extension
